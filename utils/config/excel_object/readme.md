### 建议配置生成 json，而不是直接加载excel数据
 * 因为json到程序结构数据类型基本上已经确定，如果配置有格式方便问题就可<br>
   以在从excel生成json时发现,服务器，客户端程序加载时间比较长，这样子可以提高效率
 * json相对excel文件小，可以降低服务器硬件成本要求
 * json在redis,mongo等一些组件都可以存储，为配置热更方便提供了更多选择
